# 前端简略说明（v2.0）

## 1. 这个产品想做什么

- 目标：做一个“高中错题闭环系统”，不是只给答案，而是定位错因并安排后续训练。
- 核心闭环：上传错题 -> AI诊断 -> 分流训练 -> 掌握度更新 -> 预测分变化 -> 继续推荐学习。

## 2. 产品最想要的功能（前端视角）

- 错题上传与归档：作业/考试/极简上传，支持“待诊断/已完成”状态。
- AI分层诊断：通过对话把错误定位到层级（如建模/列式/执行）。
- 定向训练：
- 模型训练：6步流程（识别、决策、列式、陷阱、完整求解、变式）。
- 知识点学习：5步流程（概念呈现、理解检查、辨析、应用、检测）。
- 掌握度体系：L0-L5（知识点和模型各一套语义）。
- 高考卷子视角：按题号看“能拿几分”，做卷面策略（MUST/TRY/SKIP）。
- 预测中心：显示目标分、当前预测、提分路径、优先训练模型。
- 记忆系统：闪卡复习（SM-2节奏），训练后自动生成卡片。
- 周复盘：本周闭环、进展、下周重点。

1. 先把“闭环主链路”打通：`index -> question-detail -> ai-diagnosis -> model-training/knowledge-learning`。
